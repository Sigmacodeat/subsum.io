# Supported file types

Subsumio is built to ingest typical legal and tax document formats reliably.

## Supported

- PDF (text-based and scanned)
- Images: PNG, JPG

## Quality tips

- For scans: 300 DPI, straight pages, high contrast.
- Keep documents unencrypted.

## If something fails

If an upload fails or gets stuck, check:

- File size and network stability
- Encrypted PDFs (remove encryption)
- Extremely low-resolution scans
